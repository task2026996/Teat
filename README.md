# World Clock Website

## Overview
This is a simple World Clock website that displays the current time in different time zones around the world.

## Features
- Displays the current time in various international cities.
- Allows users to select different time zones.
- User-friendly interface.

## Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/task2026996/Teat.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Teat
   ```
3. Open the `index.html` file in your web browser.

- HTML
